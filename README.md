## 📖 About The Project

**ChatrFinance** web dashboard is the desktop/web interface for Iran's leading real-time financial data platform.

Users can access:

- 💰 **Live Gold & Currency Rates** (IRR-based)
- 📈 **Tehran Stock Exchange** data
- 🪙 **Cryptocurrency** prices
- 🌍 **Global Stocks** (NVIDIA, Apple, etc.)
- 🛢️ **Commodities** (Oil, Gold spot)
- 💱 **Forex Rates** (USD/EUR/GBP to IRR)
- 📊 **TradingView Charts** for technical analysis
- 🗞️ **Persian Financial News**
- ✅ **Verified Analyst Posts** with admin approval
